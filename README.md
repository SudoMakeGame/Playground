Mike's Lua Playground

Go through the tutorials to create a simple Lua demo.

Reason: To learn Lua.

* http://love2d.org/
* https://code.google.com/p/luaforwindows/

### Tutorials

http://www.love2d.org/wiki/Category:Tutorials

### How-To
